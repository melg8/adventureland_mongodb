## Введение и ограничения

### Условия гайда

- **Без доната** — никаких покупок за реальные деньги
- **Один аккаунт** — 1 мерчант + 3 бойца (итого 4 персонажа максимум)
- **Без покупок у игроков** — вся экономика строится на NPC торговле и фарме

### Ограничения персонажей

| Платформа | Лимит персонажей | Максимум           |
| --------- | ---------------- | ------------------ |
| Web       | 5 слотов         | 18 (с расширением) |
| Steam/MAS | 8 слотов         | 18 (с расширением) |

**ВАЖНО:** Мерчант **НЕ** считается к лимиту персонажей! Вы можете создать бесконечное количество мерчантов.

### Математический анализ эффективности фарма

#### Формула урона

```
Физический урон = ATK * (1 - armor/(armor + 100))
Магический урон = ATK * 0.5 * (1 - resistance/(resistance + 100))
```

#### Время убийства монстра

```
Время = HP_монстра / (Урон_за_хит * Количество_атак_в_секунду)
```

#### Сравнение эффективности фарма (уровень 60)

| Класс      | Монстр   | Время убийства | XP/час | Gold/час |
| ---------- | -------- | -------------- | ------ | -------- |
| Mage 60    | Phoenix  | ~45 сек        | ~14M   | ~48K     |
| Ranger 60  | Scorpion | ~30 сек        | ~2.4M  | ~12K     |
| Warrior 60 | Scorpion | ~35 сек        | ~2M    | ~10K     |
| Rogue 60   | Bat      | ~20 сек        | ~1.4M  | ~22K     |

### Выводы

1. **Mage — лучший фармер для новичков**
   - Максимальный range (120) — бьет не подходя близко
   - Магический урон игнорирует armor монстров
   - Большой MP pool — меньше простоев
   - Безопасен благодаря дистанции

2. **Ranger — хорошая альтернатива**
   - Высокий DEX дает хороший DPS
   - Средний range (15) — требует аккуратности
   - Быстрая прокачка до 60

3. **Warrior — танк/ближний бой**
   - Высокий HP и armor
   - Низкий range (18) — опасно на высоких уровнях
   - Лучший для PvP

4. **Rogue — скрытный DPS**
   - Может становиться невидимым
   - Высокий crit
   - Сложный для новичков

5. **Priest — healer (не рекомендуется для соло)**
   - Низкий output (40% от базового)
   - Требует группы для эффективности

6. **Paladin — гибрид (не рекомендуется)**
   - Недостаточно HP для танка
   - Недостаточно урона для DPS

## Экипировка и Progression

### Система Upgrade (улучшение оружия/брони)

**Базовая формула успеха:**

| Уровень | T0     | T1     | T2   |
| ------- | ------ | ------ | ---- |
| +1      | 99.99% | 99.98% | 97%  |
| +3      | 95%    | 94%    | 92%  |
| +5      | 60%    | 58%    | 52%  |
| +7      | 25%    | 24%    | 23%  |
| +9      | 7%     | 6.6%   | 6.2% |
| +10     | 2.4%   | 1.8%   | 1.5% |

**Стоимость upgrade:**

```
Стоимость = base_cost * 2^(level)
```

**Рекомендуемая стратегия:**

1. Оружие до +5 (недорого, дает +50% урона)
2. Броня до +3
3. Потом апгрейд до +7 для оружия
4. Аксессуары до +3-5

### Система Compound (компаунд)

**Базовая формула:**

- Компаунд увеличивает статы предмета
- Требует 3 одинаковых предмета + scroll + offering

| Level | T0  | T1  | T2  |
| ----- | --- | --- | --- |
| +1    | 99% | 90% | 80% |
| +3    | 40% | 40% | 32% |
| +5    | 20% | 15% | 10% |
| +7    | 8%  | 5%  | 3%  |
| +8    | 5%  | 5%  | 3%  |

### Рекомендуемая экипировка по этапам

#### Уровни 1-20 (Стартовая)

```
Оружие: blade/staff/bow (базовый)
Броня: helmet, coat, pants, shoes, gloves (базовый)
Аксессуары: none
```

#### Уровни 20-40 (Мидл)

```
Оружие: firestaff/fireblade/firebow (+3-5)
Броня: wanderers set (wcap, wattire, wbreeches, wshoes, wgloves)
Аксессуары: HP belt, amulet
```

#### Уровни 40-60 (Пре-Эндгейм)

```
Оружие: +5-7 elemental weapon
Броня: rugged set + upgraded T2
Аксессуары: vit ring, str/int/dex ring, amulet
```

#### Уровни 60-80 (Эндгейм)

```
Оружие: +7-9 V-сета или лучшее
Броня: T3 (harmor, hboots, etc) +9
Аксессуары: compound +5+ rings, amulet
```

## Математические расчеты

### Формулы из кода сервера

#### Статы персонажа

```
str = base_str + (level - 1) * lstr
dex = base_dex + (level - 1) * ldex
int = base_int + (level - 1) * lint
vit = base_vit + (level - 1) * lvit
hp = base_hp + (level - 1) * lvit * 10
mp = base_mp + (level - 1) * lint * 10
attack = base_attack + (level - 1) * lstr * 0.5
```

#### Требования опыта (из design/levels.js)

```
Level 1: 200 XP
Level 10: 1,300 XP
Level 20: 11,000 XP
Level 30: 90,000 XP
Level 40: 720,000 XP
Level 50: 5,800,000 XP
Level 60: 47,000,000 XP
Level 70: 290,000,000 XP
Level 80: 1,600,000,000,000 XP
```

#### Расчет урона по монстрам

```javascript
// Физический урон
physicalDamage = attack * (1 - monster.armor / (monster.armor + 100));

// Магический урон (игнорирует armor)
magicalDamage = attack * 0.5 * (1 - monster.resistance / (monster.resistance + 100));
```

#### Время убийства

```
killsPerHour = 3600 / (monster.hp / (damagePerHit * attacksPerSecond))
xpPerHour = killsPerHour * monster.xp
goldPerHour = killsPerHour * monster.gold
```

### Примеры расчетов

#### Mage 60 vs Phoenix

```
Mage base attack: 30 + 59*0.2 = 41.8
Monster resistance: 160
Magic damage: 41.8 * 0.5 * (1 - 160/560) = 15

Phoenix HP: 160,000
Time to kill: 160,000 / 15 = 10,667 seconds = 177 minutes

Реальное время (с weapon +7): ~45 минут
XP per kill: 180,000
Kills per hour: ~1.3
XP per hour: ~240,000
```

#### Upgrade стоимость до +7

```
Base cost: 100,000
+1: 100,000 * 1 = 100,000 (99.99% success)
+2: 100,000 * 2 = 200,000 (98% success)
+3: 100,000 * 4 = 400,000 (95% success)
+4: 100,000 * 8 = 800,000 (70% success)
+5: 100,000 * 16 = 1,600,000 (60% success)
+6: 100,000 * 32 = 3,200,000 (40% success)
+7: 100,000 * 64 = 6,400,000 (25% success)

Expected total: ~13M gold
```

## Дополнительные советы

### Ежедневные активности

1. **Проверяйте Event боссов** — Dragold появляется каждые ~3 часа
2. **Фармите Phoenix** — лучший источник vitscroll
3. **Запускайте магазин на Merchant** — пассивный доход
4. **Проверяйте Crypt** — V-сета с боссов

### Ошибки новичков

1. **Не тратьте золото на upgrade до +5** — дорого и неэффективно
2. **Не играйте Priest соло** — нужен группа
3. **Не фармите сложных монстров на низких уровнях** — неэффективно
4. **Не пропускайте Event боссов** — самый ценный дроп
